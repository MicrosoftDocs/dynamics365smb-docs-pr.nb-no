## <a name="to-move-components-to-an-operation-area-in-basic-warehouse-configurations"></a><span data-ttu-id="51c3f-123">Flytte komponenter til et operasjonsområde i enkle lageroppsett</span><span class="sxs-lookup"><span data-stu-id="51c3f-123">To move components to an operation area in basic warehouse configurations</span></span>  
1.  <span data-ttu-id="51c3f-124">Velg ikonet ![lyspære som åpner Fortell meg-funksjonen](media/ui-search/search_small.png "Fortell hva du vil gjøre"), angi **Lagerflytting**, og velg deretter den relevante koblingen.</span><span class="sxs-lookup"><span data-stu-id="51c3f-124">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Inventory Movement**, and choose the relevant link.</span></span>  
2.  <span data-ttu-id="51c3f-125">På hurtigfanen **Generelt** fyller du ut **Nr.**-feltet</span><span class="sxs-lookup"><span data-stu-id="51c3f-125">On the **General** FastTab, fill in the **No.**</span></span> <span data-ttu-id="51c3f-126">.</span><span class="sxs-lookup"><span data-stu-id="51c3f-126">field.</span></span> <span data-ttu-id="51c3f-127">Du kan trykke ENTER for å velge fra nummerserien.</span><span class="sxs-lookup"><span data-stu-id="51c3f-127">You can press the Enter key  to select from the number series.</span></span>  
3.  <span data-ttu-id="51c3f-128">Angi lokasjonen der flyttingen finner sted, i **Lokasjonskode**-feltet.</span><span class="sxs-lookup"><span data-stu-id="51c3f-128">In the **Location Code** field, enter the location where the movement occurs.</span></span>  
4.  <span data-ttu-id="51c3f-129">Velg handlingen **Hent kildedokumenter**.</span><span class="sxs-lookup"><span data-stu-id="51c3f-129">Choose the **Get Source Documents** action.</span></span> <span data-ttu-id="51c3f-130">Du kan også fylle ut feltet **Kildedokument** og deretter velge **AssistEdit**-knappen i feltet **Kildenr.**</span><span class="sxs-lookup"><span data-stu-id="51c3f-130">Alternatively, fill in the **Source Document** field, and then choose the **AssistEdit** button in the **Source No.** field.</span></span>  
5.  <span data-ttu-id="51c3f-131">Velg monteringsordren du vil flytte komponenter for, på siden **Kildedokumenter**, og klikk deretter **OK**.</span><span class="sxs-lookup"><span data-stu-id="51c3f-131">On the **Source Documents** page, select the assembly order that you want to move components for, and then choose the **OK** button.</span></span>  

    <span data-ttu-id="51c3f-132">For hver nødvendige komponent som kan flyttes, genereres én Hent-linje og én Plasser-linje på siden **Lagerflyttinger**.</span><span class="sxs-lookup"><span data-stu-id="51c3f-132">For each needed component that can be moved, one Take line and one Place line are generated on the **Inventory Movements** page.</span></span> <span data-ttu-id="51c3f-133">Alle felt bortsett fra **Ant. som skal håndt.** er forhåndsutfylt i henhold til kildedokumentlinjene.</span><span class="sxs-lookup"><span data-stu-id="51c3f-133">All fields except the **Qty. to Handle** field are prefilled according to the source document lines.</span></span> <span data-ttu-id="51c3f-134">Feltet **Ant. som skal håndt.** settes til null inntil du angir antallet du faktisk har flyttet.</span><span class="sxs-lookup"><span data-stu-id="51c3f-134">The **Qty. to Handle** field is set to zero until you enter the quantity that you have actually moved.</span></span>  

    <span data-ttu-id="51c3f-135">Du kan endre hyllekoden på en Hent-linje, men bare i henhold til tilgjengelighet.</span><span class="sxs-lookup"><span data-stu-id="51c3f-135">You can change the bin code on a Take line but only according to availability.</span></span> <span data-ttu-id="51c3f-136">Hvis du klikker **AssistEdit**-knappen i **Hyllekode**-feltet på en Hent-linje, åpnes **Hylleinnhold**-siden, der bare hyller hvor komponenten er tilgjengelig, vises.</span><span class="sxs-lookup"><span data-stu-id="51c3f-136">If you choose the **AssistEdit** button in the **Bin Code** field on a Take line, then the **Bin Contents** page opens and only shows bins where the component is available.</span></span>  

    <span data-ttu-id="51c3f-137">Du kan ikke endre hyllekoden på en Plasser-linje.</span><span class="sxs-lookup"><span data-stu-id="51c3f-137">You cannot change the bin code on a Place line.</span></span> <span data-ttu-id="51c3f-138">Det er bare hyllekoden som er definert på komponentlinjen i kildedokumentet som godtas.</span><span class="sxs-lookup"><span data-stu-id="51c3f-138">Only the bin code that is defined on the component line of the source document is accepted.</span></span> <span data-ttu-id="51c3f-139">Dette støtter prinsippet om at rollen som ber om en komponent, som er en monteringsarbeider i denne prosedyren, vet hvor komponenten må plasseres.</span><span class="sxs-lookup"><span data-stu-id="51c3f-139">This supports the principle that the role who requests a component, which is an assembly worker in this procedure, knows where the component must be placed.</span></span> <span data-ttu-id="51c3f-140">Hvis du vil plassere komponentene på en annen hylle, må du først endre hyllekoden på komponentlinjen og deretter opprette lagerflyttingslinjene på nytt.</span><span class="sxs-lookup"><span data-stu-id="51c3f-140">If you want to place the components in a different bin, then you must first change the bin code on the component line and then re-create the inventory movement lines.</span></span>  
6.  <span data-ttu-id="51c3f-141">Angi angi det hele eller delvise antallet du faktisk har flyttet, i feltet **Ant. som skal håndt.**.</span><span class="sxs-lookup"><span data-stu-id="51c3f-141">In the **Qty. to Handle** field, enter the full or partial quantity that you have actually moved.</span></span> <span data-ttu-id="51c3f-142">Verdiene på Hent- og Plasser-linjene må være like.</span><span class="sxs-lookup"><span data-stu-id="51c3f-142">The value on the Take and the Place lines must be the same.</span></span> <span data-ttu-id="51c3f-143">Eller kan du ikke registrere flyttingen.</span><span class="sxs-lookup"><span data-stu-id="51c3f-143">Otherwise, you cannot register the movement.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="51c3f-144">Som i andre lageraktiviteter kan du dele Plasser-linjen ved å velge handlingen **Handlinger** og deretter **Del linje**.</span><span class="sxs-lookup"><span data-stu-id="51c3f-144">As in other warehouse activities, you can split the Place line by selecting the **Actions** action and then choosing the **Split Line** action.</span></span> <span data-ttu-id="51c3f-145">I dette tilfellet må summen av to delte Plasser-linjene være lik antallet på Hent-linjen.</span><span class="sxs-lookup"><span data-stu-id="51c3f-145">In that case, the sum of the two split Place lines must equal the quantity on the Take line.</span></span>  

7.  <span data-ttu-id="51c3f-146">Når du er klar til å registrere flyttingene som du har utført, velger du handlingen **Registrer lagerflytting**.</span><span class="sxs-lookup"><span data-stu-id="51c3f-146">When you are ready to register the movements that you have performed, choose the **Register Invt. Movement** action.</span></span>  

    <span data-ttu-id="51c3f-147">Lagerposter opprettes og gjenspeiler at komponentene nå er i hyllene som er angitt på monteringsordrelinjene.</span><span class="sxs-lookup"><span data-stu-id="51c3f-147">Warehouse entries are created reflecting that the components now exist in the bins specified on the assembly order lines.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="51c3f-148">I motsetning til når du flytter komponenter en lagerplukking, bokføres ikke forbruk når du registrerer en lagerflytting.</span><span class="sxs-lookup"><span data-stu-id="51c3f-148">Unlike when you move components with an inventory pick, consumption is not posted when you register an inventory movement.</span></span> <span data-ttu-id="51c3f-149">Dette trinnet må utføres separat ved å bokføre avgang og forbruk av monteringsordren.</span><span class="sxs-lookup"><span data-stu-id="51c3f-149">That step must be performed separately by posting the assembly order output and consumption.</span></span> <span data-ttu-id="51c3f-150">Hvis du vil ha mer informasjon, kan du se Monteringsordre.</span><span class="sxs-lookup"><span data-stu-id="51c3f-150">For more information, see Assembly Order.</span></span>
