<span data-ttu-id="c5085-108">Du bruker vinduet **Oppsett for bunke-ikon** til å definere indikatorer for alle bunke-ikonene som er tilgjengelige i selskapsdatabasen.</span><span class="sxs-lookup"><span data-stu-id="c5085-108">From the **Cue Setup** window, you set up indicators for all the Cues that are available in the company database.</span></span> <span data-ttu-id="c5085-109">Du kan definere indikatorene skal gjelde for alle brukere i firmaet eller en individuell bruker.</span><span class="sxs-lookup"><span data-stu-id="c5085-109">You can set up the indicators to apply to all users in the company or an individual user only.</span></span> <span data-ttu-id="c5085-110">Indikatorinnstillingene i vinduet **Oppsett for bunke-ikon** fungerer som standardinnstillingene for indikatoren.</span><span class="sxs-lookup"><span data-stu-id="c5085-110">The indicator settings in the **Cue Setup** window act as the default indicator settings.</span></span> <span data-ttu-id="c5085-111">Hvis vinduet **Oppsett for bunke-ikon, sluttbruker** gjøres tilgjengelig for brukere, kan de tilpasse innstillingene for indikatorer du definerer i vinduet **Oppsett for bunke-ikon**.</span><span class="sxs-lookup"><span data-stu-id="c5085-111">If the **Cue Setup End User** window is made available to users, then they can personalize the indicators settings that you define in the **Cue Setup** window.</span></span>
